:: คู่มือการติดตั้ง ::


  การติดตั้งใน Host จริง (ในที่นี้ใช้ของ Host ผมให้ดูนะครับ)

1. HOST ทุกที่ต้องมีตัว Control Panel ให้เข้าใช้ครับ โดยจะต้องบอกเรามาตอนจ่ายเงินครับ
(ถ้าไม่ทราบให้ถามไปที่ HOST ที่คุณใช้เลยครับ)



2. พอเข้าไปแล้ว ให้หาหัวข้อที่เกี่ยวกับ MySQL ให้ได้ครับ แล้วก็เข้าไปสร้างฐานข้อมูลใหม่ขึ้นมาครับ
พร้อมกับตั้ง Username และ Password ในการเข้าใช้ฐานข้อมูลด้วยครับ


หาหัวข้อเกี่ยวกับ MySQL ให้ได้ แล้วเข้าไป


สร้างฐานข้อมูลอันใหม่ขึ้นมา พร้อมใส่ Username และ Password ในการติดต่อ Database ด้วย


พอกดสร้างแล้ว ถ้าสำเร็จก็จะรายงานผลออกมาว่าสำเร็จ

3. กลับมาที่ไฟล์ภายในโฟลเดอร์ meeting ครับ ใช้โปรแกรม Notepad เปิดไฟล์ "config.php" ครับ
แล้วก็เข้าไปแก้ไขข้อมูลในนี้



4. เข้าไปแก้ข้อมูลในไฟล์นี้ครับ เสร็จแล้ว Save ด้วย



5. กลับมาที่ไฟล์ภายในโฟลเดอร์ meeting ครับ ใช้โปรแกรม Notepad เปิดไฟล์ "dbconfig.php"
แล้วก็เข้าไปแก้ไขข้อมูลในนี้



6. เข้าไปแก้ข้อมูลในไฟล์นี้ครับ เสร็จแล้ว Save ด้วย



7. ทำการ Upload ไฟล์ทั้งหมดขึ้นไปบน Host ของคุณเลยครับ



8. ถ้า Server ที่คุณใช้เป็น UNIX/LINUX ให้ทำการ CHMOD โฟลเดอร์ที่ชื่อ room_picture
เป็น 777 ด้วย ครับ สำคัญมากนะครับ



9. ใช้ Browser เปิดหน้า setup.php เพื่อทำการติดตั้งระบบ (เช่นของผมก็ใช้เป็น
http://www.thamwebsite.com/meeting/setup.php น่ะครับ คือขึ้นอยู่
กับว่า คุณจะ Upload ไปไว้ที่ใดน่ะครับ) โดยถ้าได้ติดตั้งสำเร็จก็จะมีข้อความ
บอกว่า "การสร้างตารางลงฐานข้อมูลเสร็จสมบูรณ์ !!"





10. เป็นอันเสร็จสิ้นการติดตั้งครับ สามารถใช้งานได้เลย
( เช่น หน้าระบบ สามารถเข้าผ่านทาง http://www.thamwebsite.com/meeting ครับ )



11. แค่นี้ก็พร้อมใช้งานแล้วครับ

12. ผู้ดูแลระบบใช้ Username : admin และ Password : admin ในการเข้าจัดการระบบทั้งหมดได้

13. ผู้ใช้งานทั่วไปใช้ Username : user และ Password : user ในการเข้าจองห้องประชุมได้
