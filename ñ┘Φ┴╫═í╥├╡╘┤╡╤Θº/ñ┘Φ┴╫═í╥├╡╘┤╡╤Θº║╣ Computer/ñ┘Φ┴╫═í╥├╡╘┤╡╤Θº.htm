:: คู่มือการติดตั้ง ::


  การติดตั้งใน Host จำลอง โดยการใช้โปรแกรม Appserv

1. ดาวน์โหลดโปรแกรม Appserv มาติดตั้งให้เรียบร้อยก่อน คลิ๊กที่นี่เพื่อดาวน์โหลด !

2. ส่วนวิธีการติดตั้ง คลิ๊กที่นี่ครับ ! (ใส่ค่าตามรูปภาพนั้นไปเลยครับ)

3. นำโฟลเดอร์ชื่อ "meeting" ไปวางใน "C:\AppServ\www" ครับ



4. ใช้ Browser เปิดหน้า http://localhost/phpmyadmin แล้วจะมีให้กรอก Username และ Password เพื่อ
เข้า ก็ให้กรอกอันที่ตอนใช้ติดตั้งลงไปครับ (โดย Default แล้ว Username = root , Password = ใช้ที่ตั้ง
ไว้ตอนติดตั้ง นะ) พอเข้ามาได้แล้วจะเป็นแบบด้านล่างครับ



5. ให้สร้างฐานข้อมูลใหม่ชื่อ "meeting" (จะใช้ชื่ออื่นก็ได้) โดยการเรียงลำดับควรจะ
ให้เป็น "utf8_bin"



6. กลับมาที่ไฟล์ภายในโฟลเดอร์ meeting ครับ ใช้โปรแกรม Notepad เปิดไฟล์ "config.php" ครับ
แล้วก็เข้าไปแก้ไขข้อมูลในนี้ [ใช้ข้อมูลเหมือนที่ตั้งในข้อ 4-5]



7. เข้าไปแก้ข้อมูลในไฟล์นี้ครับ เสร็จแล้ว Save ด้วย



8. กลับมาที่ไฟล์ภายในโฟลเดอร์ meeting ครับ ใช้โปรแกรม Notepad เปิดไฟล์ "dbconfig.php"
แล้วก็เข้าไปแก้ไขข้อมูลในนี้



9. เข้าไปแก้ข้อมูลในไฟล์นี้ครับ เสร็จแล้ว Save ด้วย



10. ใช้ Browser เปิดหน้า http://localhost/meeting/setup.php เพื่อทำการติดตั้งระบบ ถ้าสำเร็จก็จะมี
ข้อความบอกว่า "การสร้างตารางในการใช้งานสำเร็จ !!"





11. เป็นอันเสร็จสิ้นการติดตั้งครับ สามารถใช้งานได้เลย
( เช่น หน้าระบบ สามารถเข้าผ่านทาง http://localhost/meeting ครับ )



12. ผู้ดูแลระบบใช้ Username : admin และ Password : admin ในการเข้าจัดการระบบทั้งหมดได้

13. ผู้ใช้งานทั่วไปใช้ Username : user และ Password : user ในการเข้าจองห้องประชุมได้
